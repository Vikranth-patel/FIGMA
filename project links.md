🚂 IRCTC App UX/UI Redesign Case Study

📌 Project Overview

This project is an end-to-end UX/UI redesign of the Indian Railway Catering and Tourism Corporation (IRCTC) mobile application. The goal of this project is to solve notoriously clunky interface issues, streamline the booking process, and eliminate severe friction points like payment failures and navigation clutter.

Every design decision in the final user interface is backed by real user evidence gathered through a structured, four-step design pipeline.


🛠️ Step 1: User Research & Interviews

To avoid designing on assumptions, the project began with qualitative and quantitative user research. I conducted user interviews with 5 real users using 5-7 open-ended questions to capture genuine frustrations. This was supplemented by a broader digital survey to quantify the pain points.

Targeted Platform: IRCTC Native Mobile App (iOS/Android)

Core Tasks Evaluated: Tatkal booking, general ticket search, and payment flow.

🗺️ Step 2: Affinity Mapping (Synthesis)

After gathering raw data from the interviews and surveys, I utilized a digital collaborative space to dump my notes onto sticky notes and group them into organic themes. This helped pinpoint the most critical areas requiring a redesign.

Identified Themes: Payment Failures, App Performance & Stability, Navigation & UI Clutter, Login & Authentication Issues.

👤 Step 3: User Persona Construction

Using the synthesized data from the affinity map, I built a core user persona to serve as the "North Star" for the UI redesign. The persona template covers Bio, Core Needs, Frustrations, Personality Tags, and Technical Proficiency.

Primary Persona: Rajiv Kumar, The Anxious Commuter

Core Needs: Fast checkout, accurate real-time updates, and securing Tatkal tickets without app crashes.

Key Frustrations: Double payment deductions, frequent random logouts, and severe visual clutter.

📱 Step 4: High-Fidelity UI Redesign

The final phase translates the researched pain points and persona requirements into a modern, cohesive user interface. The new screens dismantle the archaic legacy architecture, utilizing clean visual hierarchies, accessible contrast ratios, and a native, seamless checkout experience.
