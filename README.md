📖 About This Project

This project is a comprehensive UX/UI case study focused on redesigning the Indian Railway Catering and Tourism Corporation (IRCTC) mobile application. As a platform that powers one of the busiest transportation networks in the world, the IRCTC ecosystem handles millions of daily users and processes roughly 800,000 online tickets every day . However, the current mobile interface is notoriously clunky, suffering from severe visual clutter, system instability during peak "Tatkal" booking hours, and a highly disjointed payment flow that frequently frustrates users.

The primary objective of this project is to showcase an evidence-based redesign of the IRCTC booking experience. Rather than designing based on assumptions, this project strictly follows a user-centered design pipeline to ensure every screen built is backed by real user evidence:

1. User Research: The project began by capturing real-world friction points through digital surveys and conducting 5 direct user interviews. By asking 5-7 carefully crafted open-ended questions, I was able to talk to real people and uncover their exact pain points during the booking process.

2. Data Synthesis (Affinity Mapping): Raw qualitative data and interview notes were dumped onto digital sticky notes in FigJam. These notes were then organically grouped into overarching themes—such as "Payment Failures," "App Instability," and "Visual Clutter"—to identify the most critical areas requiring a redesign.

3. Persona Construction: To bring the target user to life, the synthesized data was used to build a detailed User Persona using a comprehensive template in Figma. This profile outlines the user's core demographics, technical literacy, payment mediums, and a narrative bio, while clearly defining their core needs and highest frustrations.

4. Evidence-Backed UI Design: Armed with clear insights and an empathetic persona, the project culminates in a high-fidelity user interface redesign. The new screens dismantle the archaic legacy architecture, introducing a clean visual hierarchy, seamless navigation, and a modern checkout experience to solve the exact problems identified during research.

This repository serves as a complete project hub, containing links to the raw survey data, the FigJam affinity mapping board, the Figma persona blueprints, and the final high-fidelity UI design files.
